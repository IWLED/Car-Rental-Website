
 منصة تأجير السيارات
 
منصة تأجير سيارات تربط بين التجار والعملاء، وتوفر نظام بسيط وسلس لإدارة عمليات الحجز والتأجير.

المميزات:

العملاء:
تسجيل الدخول وإنشاء حساب
تصفح السيارات المتاحة مع التصفية
اختيار مدة الإيجار
الدفع الإلكتروني
التجار:
إدارة ومتابعة الطلبات
إدارة بيانات المستخدمين وسجل الطلبات
تعديل بيانات الطلبات والمستخدمين

طريقة التشغيل:

تثبيت XAMPP أو WAMP
استيراد ملف قاعدة البيانات (.sql)
نقل المشروع إلى مجلد htdocs
تشغيل المشروع عبر المتصفح على localhost

مشروع بسيط قابل للتطوير والتوسعة.
